# 项目 96933223677：数字人与旁白同时开口、字幕难对照问题说明

**Langfuse trace**：`analysis/langfuse-data/cases/96933223677/trace-1-5344de98.json`（同目录另有 `trace-2`～`trace-4`；体积较大。）  
**分析日期**：2026-05-05  

---

## 一、用户侧现象

- **听感**：数字人（口型/画面）与 **TTS 旁白** 像**同时在说话**，两路人声糊在一起。  
- **观感**：底部烧录字幕**难以与数字人「说的话」在字级、节奏上对齐**（有时像对 TTS，有时像对不上嘴）。  

---

## 二、根因（本 trace 可核对）

### 1. 双路人声：`sound: "on"` 又叠了 ElevenLabs VO

- 三段 `video_generate`（Seedance）均为 **`sound: "on"`**，成片自带模型生成的**环境声 + 可能含糊的「说话感」同期声**。  
- 同时又 `audio_produce` 生成三段 **VO（ElevenLabs）**。  
- `execute_edit_video` 中 **视频轨每个 clip 的 `volume` 为 `0.35`** → **未关掉成片音轨**，与 **满音量 VO**、BGM 混音 → 听感上即「数字人轨 + 旁白轨」并行。

与 **script-skill** 中 **「每条口播只有一个音源（co-gen 或 TTS，勿双开）」** 的约束不一致。

### 2. 字幕与「嘴」不对齐：锚定对象不同

| 对象 | 实际对齐的是 |
|------|----------------|
| **字幕时间轴** | **TTS** 时长 + Agent 手拆的 `text` cue（未强制以 `subtitle_file` 为唯一真源）。 |
| **口型与身体语言** | **Seedance 15s/10s 视频** 自己的节奏；**未**把 TTS 作为 `audio_list` 等条件输入做口型锁定。 |
| **成片里剩的 35% 视频声** | 若模型自造语声，与字幕文案也可能不一致。 |

因此：**字幕 = 对齐旁白文案 + TTS**；**画面 = 另一套生成结果** → 无法要求「字幕完全对照数字人的话」除非改产线（见第五节）。

---

## 三、与「字幕重叠」（14194336910 类 case）的区分

- **96933223677**：主要是 **音轨叠两路人声 + 口播真源分裂**（本节第二节）。  
- **14194336910**：主要是 **`text` 轨时间窗相交或手填时间与 TTS 不一致**（见 `case-14194336910-字幕重叠问题说明.md`）。  
两类问题可并存，但**修法不同**：本节第四节先关双声道白，再谈字幕真源与口型策略。

---

## 四、可落地修复（成片 / 再生一版）

1. **有独立 VO 时**：对应 `video_generate` 使用 **`sound: "off"`**（或等价关闭成片人声/同期声）。  
2. **合成阶段**：视频轨 **`volume: 0`**（或丢弃视频音轨），**人声只来自 VO 轨**；环境声用单独 SFX/设计轨。  
3. **字幕**：优先用 `audio_produce` 返回的 **`subtitle_file`** 驱动 `text`/`subtitle` 轨，文案与 TTS `text` 一致；避免纯手拍毫秒。  

---

## 五、若产品要求「口型必须跟这句英文」

- **不能**长期依赖「有声 Seedance + 再叠同文案 TTS」而不做音画耦合。  
- 应改为：**口播驱动画面**（如每段 TTS/干声进 `audio_list` 条件生成），或 **单段 co-gen**：台词仅在 prompt 内、**`sound: "on"` 且该段不再叠 TTS**。  
- 若坚持 **VO + 数字人仅形象**：应减少「正对镜头长时间讲话」镜头，改手势、看 UI、侧脸等，降低对口型预期。

---

## 六、对 skill 的条文化建议（与 14194336910 文档互补）

- **generation-skill / audio-strategy**：若已规划 **`audio_produce` 承担该段全部口播** → 对应 **`video_generate` 必须 `sound: "off"`**。  
- **assembly-skill**：存在独立 VO 轨时，**默认视频轨音量为 0**；仅当「成片人声即唯一口播且无 TTS」时允许保留视频音轨。  
- **script-skill**：用户写明 **Voiceover & Subtitles** 时，handoff 中标注 **「仅 TTS + Layer2 字幕」**，并写清 **禁止** 同段 Seedance `sound: on` 带对白。
