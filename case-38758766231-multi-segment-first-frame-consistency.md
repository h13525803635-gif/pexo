# Case 38758766231 — 多段视频首帧一致性问题

**需求**：美甲 UGC 广告（人物+参考视频+美甲样式，15s 竖屏）  
**时间**：2026-04-20 13:13 / 2026-04-21 03:21–03:32（3 个 trace）

---

## 项目概况

Session 内两个独立任务：

1. **Trace-1**：泳衣展示大片（模特+比基尼，15s 竖屏）→ 生成 A/B 两个预览，无首帧问题
2. **Trace-2/3**：美甲 UGC 广告 → Trace-2 生成 fast 概念预览；Trace-3 用 Seedance 正式版分 3 段生成后合成 → **首帧一致性问题集中在此**

---

## 根因：3 段独立生成，无首帧/末帧锁定

Trace-3 三段视频参数：

| 段 | 时长 | image_list | first_frame | last_frame |
|----|------|-----------|-------------|------------|
| `nail_ugc_seg1`（0–4s，正脸开场） | 4s | `[a_31vdnWr]`（仅人物图） | **无** | **无** |
| `nail_ugc_seg2`（4–11s，美甲特写） | 7s | `[a_31vdnWr, a_b592vRD]` | **无** | **无** |
| `nail_ugc_seg3`（11–15s，正脸收尾） | 4s | `[a_31vdnWr, a_b592vRD]` | **无** | **无** |

三段全部使用 `reference2video` 模式，**无任何一段传入 `first_frame` 或 `last_frame`**。Seedance 支持首帧锁定，但完全未使用。

### 具体影响

- **seg1 末帧 → seg2 首帧**：人物位置、光线、背景完全独立采样，拼接处跳变
- **seg2 末帧 → seg3 首帧**：从手部特写切回正脸无任何视觉桥接
- **seg1 vs seg3 人物一致性**：两段都是正脸自拍，但独立生成，同一人的发位、表情、光线角度各异，人物外貌可能漂移

### 次要问题：image_list 在各段不一致

seg1 只传人物参考图，seg2/seg3 同时传人物+美甲图。两套 reference 配置不同，Seedance 对同一人物的理解在段间存在偏差，进一步拉低人物一致性。

---

## 正确做法

```
seg1 生成后：
  提取 seg1 末帧 → 作为 seg2 的 first_frame

seg2 生成后：
  提取 seg2 末帧 → 作为 seg3 的 first_frame

所有段的 image_list 保持一致
```

Seedance `first_frame` 参数专门解决多段衔接问题。generation-skill 在多段生成时应强制执行「提取末帧 → 传入下段 first_frame」的流程，当前未执行，属于**多段拼接场景下的系统性遗漏**。
